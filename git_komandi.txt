Git komandi od vtor cas:

- git branch - is used to list all the local branches
- git branch branch-name - is used to create new branch
- git checkout branch-name - is used to switch from one branch to another branch in your repository
- git branch -d branch-name - is used to delete the specified branch
- git pull - is used to fetch new changes from a remote repository and integrate them into your local 
branch in one step.
- git stash - allows you to temporarily save changes that are not yet ready to be committed, so that 
you can work on something else without losing your progress.
- git stash apply - applies the most recent stash that you have saved to your repository.
- git clone repository-url - allows you to copy a Git repository from a remote source to your local 
machine.
